## bookkeeper-go

### Starting point for G3A to develop a bookkeeping app.

The app has 3 parts:
- **BookKeeperData** - SQL scripts for the database
- **BookKeeperBE** - Back-end part of the app - ASP.NET Core API
- **BookKeeperFE** - Front-end part of the app - Angular
